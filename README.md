# Telegram bot for Stable Diffusion neural network

## Installation

```bash
pip install .
printf "TELEGRAM_BOT_TOKEN=...\nSTABLE_DIFFUSION_PATH=..." > .env
```

## Running

```bash
python3 main.py
```
